# eslint-plugin-web

wip: Additional ESLint's rules for web.

## 💿 Install & Usage

```bash
    npm install --save-dev eslint eslint-plugin-web
```

- Requires Node.js `>=8.10.0`
- Requires ESLint `>=5.16.0`

**.eslintrc.json** (An example)

```json
{
    "extends": [
        "plugin:web/all"
    ]
}
```
